# Alex Chen - Sound Design Portfolio

A modern, responsive sound design portfolio website built with Next.js, TypeScript, and Tailwind CSS. Features a sleek dark theme with professional animations and audio-focused design elements.

## 🎵 Features

- **Modern Design**: Dark theme with high-end audio production aesthetic
- **Responsive Layout**: Fully responsive design that works on all devices
- **Smooth Animations**: Framer Motion animations for professional feel
- **Audio Portfolio**: Interactive audio players with waveform visualizations
- **Video Reel**: Showcase video projects with embedded players
- **Services Section**: Comprehensive service offerings with detailed descriptions
- **Contact Form**: Functional contact form with validation
- **Professional UI**: Glass morphism effects and gradient animations

## 🚀 Tech Stack

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Forms**: React Hook Form with Zod validation
- **Deployment**: Vercel-ready

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd sound-design-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Building for Production

```bash
npm run build
npm start
```

## 🎨 Customization

### Colors
The color scheme can be customized in `src/app/globals.css` by modifying the CSS custom properties.

### Content
Update the content in each component file to match your personal information:
- Personal details in `Header.tsx` and `Hero.tsx`
- Portfolio items in `AudioPortfolio.tsx` and `VideoReel.tsx`
- Services in `Services.tsx`
- Contact information in `Contact.tsx` and `Footer.tsx`

### Audio/Video Content
Replace placeholder content with your actual:
- Audio files and URLs
- Video thumbnails and links
- Portfolio descriptions
- Service offerings
